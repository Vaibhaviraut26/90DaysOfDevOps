# **90-Day DevOps Learning Plan**


## Goal of This Plan

To build a strong foundation in DevOps and Cloud Engineering in 90 days and become job-ready for an entry-level DevOps role.

## **My Understanding of DevOps & Cloud Engineering**

### **What is DevOps?**
DevOps is a culture and set of practices that combines Development (Dev) and Operations (Ops) to deliver software faster, reliably, and continuously.

It focuses on:

- Automation

- Continuous Integration & Continuous Deployment (CI/CD)

- Collaboration between teams

- Monitoring and reliability

### **What is Cloud Engineering?**
Cloud Engineering involves using cloud platforms (like AWS) to:

- Deploy applications

- Manage servers and storage

- Scale systems automatically

- Reduce infrastructure cost



## **Why I Am Learning DevOps & Cloud**

- I want to enter the IT/Software industry with in-demand skills

- DevOps has high job demand and growth

- I enjoy automation, Linux, and problem-solving

- Cloud removes hardware dependency and gives real-world exposure

- DevOps roles offer good career growth and learning opportunities



## **Where I Want to Reach (My Target)**

- By the end of 90 days, I want to:

- Understand DevOps concepts

- Be confident with Linux, Git, GitHub

- Deploy applications on AWS Cloud

- Build CI/CD pipelines

- Create real DevOps projects
